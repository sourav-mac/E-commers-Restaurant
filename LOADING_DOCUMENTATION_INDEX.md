## ğŸ’¡ Key Concepts

### **LoadingContext**
- Global state for loading state
- Provides `useLoading()` hook
- Manages `showLoading()` and `hideLoading()`

### **GlobalLoader**
- Reusable loader component
- Shows your truck animation
- Centers on screen with overlay

### **fetchInterceptor**
- Intercepts all `fetch()` calls
- Automatically shows/hides loader
- Handles concurrent requests

### **useLoadingState Hook**
- Helper for manual loading control
- Wraps async operations
- Auto cleanup on error
